# Production Grade E-Commerce DevOps Platform

A production-ready cloud-native e-commerce platform implementing end-to-end DevOps practices using Docker, Kubernetes, Helm, GitHub Actions, Argo CD, and AWS.

## Project Goal

To demonstrate complete DevOps workflow including:

- Containerization using Docker
- Kubernetes orchestration
- Helm package management
- CI using GitHub Actions
- CD using Argo CD
- Ingress and DNS configuration
- Cloud deployment on AWS
- End-to-End CI/CD automation

## Tech Stack

- AWS EC2
- Docker
- Kubernetes (k3s)
- Helm
- GitHub Actions
- Argo CD
- NGINX Ingress
- PostgreSQL
